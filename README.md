## 【功能介绍】
**主要功能∶系统设计了2个用户角色，分别是管理员、员工
管理员实现了入库管理、出库管理、商品管理、供应商商品管理、客户管理、供应商管理、员工管理、
库存盘点，销售统计等模块
员工拥有入库管理、出库管理、商品管理、供应商商品管理、客户管理、供应商管理的部分权限**
